## ⚠️ 개선 가능한 부분

### 1. 타 기관 배제 지시의 구체성 부족

**현재**:
```
1. **대상 한정**: 타 기관에 대한 내용은 철저히 배제하고, 오직 [대상 기관]의 평판/이미지와 관련된 표현만 추출해라.
```

**문제점**:
- "타 기관에 대한 내용"이 무엇을 의미하는지 모호할 수 있음
- 다른 기관의 특징, 장단점, 비교 언급 등이 명시적으로 나열되지 않음
- LLM이 판단하기 어려운 엣지 케이스 존재 가능

**예시 케이스**:
- "한국전력은 삼성전자보다 효율성이 낮다" → "효율성 낮음"을 한국전력 키워드로 추출하면 안 됨
- "한국가스공사는 한국전력보다 안전하다" → "안전"을 한국전력 키워드로 추출하면 안 됨

### 2. 비교/대조 상황 처리 미흡

**현재**:
```
- **긍정**: 기관의 이미지를 실질적으로 향상시키는 내용 (단, '개선 촉구'나 '타 기관 우위' 비교는 제외).
```

**문제점**:
- "타 기관 우위 비교는 제외"라고 했지만, **타 기관 열위 비교**는 명시되지 않음
- 예: "한국전력은 한국가스공사보다 비효율적이다" → 이것도 배제해야 하는데 명시 안 됨

### 3. 문장 구조 분석 지시 부족

**문제점**:
- 주어가 무엇인지 명시하지 않음
- 예: "정부가 한국전력에 압박을 가했다" → "압박"이 한국전력 키워드인지, 정부 키워드인지 모호

**참고**: 실험용 프롬프트(`prompts_experimental.py`)에는 다음이 있음:
```
2. 키워드 추출 시 **기관이 주어인 문장**에서만 추출해:
   - ✅ "한전이 신재생에너지 사업을 확대한다" → 한전 기준 긍정
   - ❌ "정부가 한전에 압박을 가했다" → 정부가 주어이므로 제외
```

### 4. 중복 제거 지시의 위치

**현재**:
```
아래 [기사]를 정밀 분석하여 [대상 기관]과 관련된 키워드를 먼저 중복된 것을 제거 한 채 추출해라
```

**문제점**:
- "중복된 것을 제거"가 언제 수행되는지 모호함
- 추출 전에 제거하는지, 추출 후에 제거하는지 불명확
