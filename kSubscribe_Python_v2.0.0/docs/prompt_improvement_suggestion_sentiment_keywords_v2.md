## 🎯 핵심 문제 이해 (수정)

### 올바른 이해

**비교 문장에서 대상 기관에 대한 직접/간접 평가 모두 추출해야 함**

| 문장 | 대상 기관 | 키워드 추출 여부 | 추출 키워드 | 이유 |
|------|----------|----------------|-----------|------|
| "한국전력은 삼성전자보다 효율성이 낮다" | 한국전력 | ✅ | "효율성 낮음" (부정) | 직접 평가 |
| "한국가스공사는 한국전력보다 효율적이다" | 한국전력 | ✅ | "비효율" 또는 "효율성 낮음" (부정) | 간접 평가 (상대적 열위) |
| "한국전력은 한국가스공사보다 안전하다" | 한국전력 | ✅ | "안전" (긍정) | 직접 평가 |
| "한국가스공사는 한국전력보다 안전하지 않다" | 한국전력 | ✅ | "안전" (긍정) | 간접 평가 (상대적 우위) |
| "삼성전자는 혁신적이다" | 한국전력 | ❌ | - | 타 기관 평가, 한국전력과 무관 |
| "한국가스공사는 효율적이다" | 한국전력 | ❌ | - | 타 기관 평가, 비교 없음 |

### 핵심 원칙

1. **직접 평가**: 대상 기관이 주어이거나 평가 대상인 경우
2. **간접 평가**: 비교 문장에서 타 기관이 주어이지만, 대상 기관에 대한 상대적 평가가 드러나는 경우
3. **타 기관 평가**: 비교 없이 타 기관만 언급된 경우는 배제

### 개선안 1: 문장 구조 분석 중심 (권장) - 수정본

```python
sentiment_keywords_improved_v1 = f"""
Reasoning: high
# Valid channels: analysis, final

너는 뉴스 기사에서 특정 기관의 평판에 영향을 미치는 핵심 키워드를 추출하는 '평판 분석 전문가'다.
아래 [기사]를 정밀 분석하여 [대상 기관]과 관련된 키워드를 먼저 중복된 것을 제거 한 채 추출해라
그 이후, 감성별로 분류해라.

### [분석 가이드라인]
1. **대상 기관 중심 추출**: 오직 [대상 기관]에 대한 평가만 추출해라.
   - ✅ 추출해야 하는 경우:
     * [대상 기관]이 주어인 문장 (예: "한국전력이 신재생에너지 사업을 확대한다")
     * [대상 기관]에 대한 직접적 평가 (예: "한국전력은 효율적이다")
     * 비교 문장에서 [대상 기관]에 대한 직접 평가 (예: "한국전력은 삼성전자보다 효율성이 낮다" → "효율성 낮음")
     * 비교 문장에서 [대상 기관]에 대한 간접 평가 (예: "한국가스공사는 한국전력보다 효율적이다" → 한국전력 키워드: "비효율" 또는 "효율성 낮음")
   - ❌ 추출하면 안 되는 경우:
     * 타 기관이 주어이고 비교가 없는 문장 (예: "삼성전자는 혁신적이다" → 한국전력 키워드로 추출 금지)
     * 타 기관에 대한 평가이고 비교가 없는 문장 (예: "한국가스공사는 효율적이다" → 한국전력 키워드로 추출 금지)

2. **비교 문장 처리 (중요)**:
   - 비교 문장에서는 [대상 기관]에 대한 직접/간접 평가 모두 추출해라.
   
   **직접 평가 (대상 기관이 주어)**:
   - 예: "한국전력은 삼성전자보다 효율성이 낮다"
     * ✅ 한국전력 키워드: "효율성 낮음" 또는 "비효율" (부정)
   
   **간접 평가 (타 기관이 주어, 대상 기관이 비교 대상)**:
   - 예: "한국가스공사는 한국전력보다 효율적이다"
     * ✅ 한국전력 키워드: "비효율" 또는 "효율성 낮음" (부정)
     * 분석: 한국가스공사가 효율적이라는 것은, 비교 대상인 한국전력은 상대적으로 비효율적이라는 의미
   
   - 예: "한국가스공사는 한국전력보다 안전하지 않다"
     * ✅ 한국전력 키워드: "안전" (긍정)
     * 분석: 한국가스공사가 안전하지 않다는 것은, 비교 대상인 한국전력은 상대적으로 안전하다는 의미
   
   - 예: "한국전력은 한국가스공사보다 안전하다"
     * ✅ 한국전력 키워드: "안전" (긍정)

3. **상호 배타성(Mutually Exclusive)**: 하나의 키워드는 오직 하나의 카테고리(긍정, 부정, 중립)에만 속해야 한다.
   - 만약 어떤 키워드가 '긍정'이나 '부정'으로 분류된다면, 절대 '중립'에 포함시키지 마라.
   - 우선순위: 긍정/부정 > 중립. (감성이 뚜렷하면 중립에서 제외)

4. **문맥 고려**:
   - **긍정**: 기관의 이미지를 실질적으로 향상시키는 내용
     * 직접 평가: "한국전력은 안전하다"
     * 간접 평가: "한국가스공사는 한국전력보다 안전하지 않다" → 한국전력은 상대적으로 안전
   - **부정**: 기관의 이미지를 실질적으로 저하시키는 내용
     * 직접 평가: "한국전력은 효율성이 낮다"
     * 간접 평가: "한국가스공사는 한국전력보다 효율적이다" → 한국전력은 상대적으로 비효율
   - **중립**: 기관과 관련되나 평판에 영향이 없는 단순 사실(일정, 통계, 공고 등)

5. **키워드 형태**: 문맥을 반영한 구체적인 표현으로 추출해라.
   - 직접 평가: "효율성 낮음", "비효율", "안전"
   - 간접 평가: 비교 문장에서 상대적 평가를 반영 (예: "비효율", "효율성 낮음", "안전")

6. **언어**: JSON의 모든 값은 반드시 한국어로 작성해라.

### [대상 기관]
이름: [organization]
(동의어: [synonyms])

### [기사]
[contents]

### [출력 형식]
반드시 아래 JSON 포맷으로만 응답해라. 주석, 설명은 절대 포함하지 마라.

{{
    "positiveKeywords": ["긍정 키워드1", "긍정 키워드2", ...],
    "negativeKeywords": ["부정 키워드1", "부정 키워드2", ...],
```

1. ✅ 간접 평가 명시 추가

## 📊 핵심 차이점 정리

### 직접 평가 vs 간접 평가

| 유형 | 예시 | 대상 기관 키워드 | 이유 |
|------|------|----------------|------|
| **직접 평가** | "한국전력은 삼성전자보다 효율성이 낮다" | "효율성 낮음" (부정) | 대상 기관이 주어, 직접 평가 |
| **간접 평가** | "한국가스공사는 한국전력보다 효율적이다" | "비효율" (부정) | 타 기관이 주어, 비교를 통해 대상 기관 상대적 평가 |
| **타 기관 평가** | "한국가스공사는 효율적이다" | 추출 안 함 | 비교 없음, 대상 기관과 무관 |

### 간접 평가 분석 방법

**패턴 1: 상대적 열위**
- "A는 B보다 우수하다" → B 키워드: "열위" 또는 해당 특성의 부정 표현
- 예: "한국가스공사는 한국전력보다 효율적이다" → 한국전력: "비효율"

**패턴 2: 상대적 우위**
- "A는 B보다 열등하다" → B 키워드: "우위" 또는 해당 특성의 긍정 표현
- 예: "한국가스공사는 한국전력보다 안전하지 않다" → 한국전력: "안전"

## 🎯 최종 권장: 개선안 1 (수정본)

### 권장 이유

1. ✅ **직접/간접 평가 구분**: 명확한 기준 제시
2. ✅ **비교 문장 처리**: 직접/간접 평가 모두 포함
3. ✅ **구체적 예시**: 간접 평가 예시와 분석 포함
4. ✅ **실용적**: 프롬프트 길이가 적절하고 이해하기 쉬움

### 핵심 메시지

1. **직접 평가**: 대상 기관이 주어이거나 평가 대상
2. **간접 평가**: 타 기관이 주어이지만, 비교를 통해 대상 기관에 대한 상대적 평가가 드러남
3. **타 기관 평가**: 비교 없이 타 기관만 언급된 경우는 배제
