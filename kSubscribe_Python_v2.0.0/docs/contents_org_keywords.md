### 주요 특징

1. **대부분의 기관이 1-2개의 키워드 보유**
   - 기관명과 약칭을 키워드로 사용하는 경우가 많음
   - 예: "한국전력공사" + "한국전력"

2. **특수 케이스**
   - **나라장터 (A0004)**: 일반 키워드 사용 (데이터, 인공지능, AI, 플랫폼)
   - **한국농수산식품유통공사 (A0029)**: 키워드 없음

3. **전력 관련 기관 그룹**
   - 한국전력공사 및 계열사들이 체계적으로 키워드 관리
   - 각 발전사들이 정식명칭과 약칭을 모두 키워드로 등록

## 📝 참고사항

- 이 데이터는 MongoDB `contents_org` 컬렉션의 `orgKeywordList` 필드에서 추출되었습니다.
- 키워드는 주로 기관명과 약칭으로 구성되어 있습니다.
- 현재 분석 시스템에서는 모든 기관에 동일한 `predefine_keyword` 리스트(29개)를 사용하고 있습니다.
- 기관별 키워드(`orgKeywordList`)는 별도로 관리되지만, 현재 분석 로직에서는 활용되지 않고 있습니다.

## 🔗 관련 컬렉션

- **predefine_keyword**: 전체 키워드 풀 (29개)
- **contents_org**: 기관 정보 및 기관별 키워드 (46개 기관)
- **predefine_keyword_relation**: 키워드 간 관계 정보
