| 문장 | 대상 기관 | 키워드 추출 여부 | 이유 |
|------|----------|----------------|------|
| "한국전력은 삼성전자보다 효율성이 낮다" | 한국전력 | ✅ "효율성 낮음" 추출 | 한국전력에 대한 부정적 평가 |
| "삼성전자는 혁신적이다" | 한국전력 | ❌ 추출 안 함 | 삼성전자에 대한 평가, 한국전력과 무관 |
| "한국전력은 한국가스공사보다 안전하다" | 한국전력 | ✅ "안전" 추출 | 한국전력에 대한 긍정적 평가 |

### 문제 상황

**타 기관의 특징/장단점이 언급되는 경우**:
- ❌ "삼성전자는 혁신적이다" → 한국전력 키워드로 "혁신" 추출하면 안 됨
- ❌ "한국가스공사는 효율적이다" → 한국전력 키워드로 "효율" 추출하면 안 됨

**비교 문장에서 대상 기관 평가**:
- ✅ "한국전력은 삼성전자보다 효율성이 낮다" → 한국전력 키워드로 "효율성 낮음" 추출해야 함
- ✅ "한국전력은 한국가스공사보다 안전하다" → 한국전력 키워드로 "안전" 추출해야 함

## 💡 개선된 프롬프트 제안

### 핵심 원칙

1. **대상 기관 중심**: 대상 기관에 대한 평가만 추출
2. **비교 문장 포함**: 비교 문장에서도 대상 기관 평가는 추출
3. **타 기관 평가 배제**: 타 기관에 대한 평가는 배제

### 개선안 1: 문장 구조 분석 중심 (권장)

```python
sentiment_keywords_improved_v1 = f"""
Reasoning: high
# Valid channels: analysis, final

너는 뉴스 기사에서 특정 기관의 평판에 영향을 미치는 핵심 키워드를 추출하는 '평판 분석 전문가'다.
아래 [기사]를 정밀 분석하여 [대상 기관]과 관련된 키워드를 먼저 중복된 것을 제거 한 채 추출해라
그 이후, 감성별로 분류해라.

### [분석 가이드라인]
1. **대상 기관 중심 추출**: 오직 [대상 기관]에 대한 평가만 추출해라.
   - ✅ 추출해야 하는 경우:
     * [대상 기관]이 주어인 문장 (예: "한국전력이 신재생에너지 사업을 확대한다")
     * [대상 기관]에 대한 직접적 평가 (예: "한국전력은 효율적이다")
     * 비교 문장에서 [대상 기관]에 대한 평가 (예: "한국전력은 삼성전자보다 효율성이 낮다" → "효율성 낮음")
   - ❌ 추출하면 안 되는 경우:
     * 타 기관이 주어인 문장 (예: "삼성전자는 혁신적이다" → 한국전력 키워드로 추출 금지)
     * 타 기관에 대한 평가 (예: "한국가스공사는 효율적이다" → 한국전력 키워드로 추출 금지)
     * 비교 문장에서 타 기관에 대한 평가 (예: "한국가스공사는 한국전력보다 효율적이다" → 한국전력 키워드로 "효율적" 추출 금지)

2. **비교 문장 처리**:
   - 비교 문장에서도 [대상 기관]에 대한 평가는 추출해라.
   - 예: "한국전력은 삼성전자보다 효율성이 낮다"
     * ✅ 한국전력 키워드: "효율성 낮음" (부정)
     * ❌ 한국전력 키워드: "효율" (X, 타 기관과의 비교이므로)
   - 예: "한국전력은 한국가스공사보다 안전하다"
     * ✅ 한국전력 키워드: "안전" (긍정)
     * ❌ 한국전력 키워드: "안전 우위" (X, 비교 표현은 제외)

3. **상호 배타성(Mutually Exclusive)**: 하나의 키워드는 오직 하나의 카테고리(긍정, 부정, 중립)에만 속해야 한다.
   - 만약 어떤 키워드가 '긍정'이나 '부정'으로 분류된다면, 절대 '중립'에 포함시키지 마라.
   - 우선순위: 긍정/부정 > 중립. (감성이 뚜렷하면 중립에서 제외)

4. **문맥 고려**:
   - **긍정**: 기관의 이미지를 실질적으로 향상시키는 내용
     * 단, '개선 촉구'나 '타 기관 우위' 비교 표현은 제외
     * 예: "한국전력은 한국가스공사보다 안전하다" → "안전" (긍정) 추출
   - **부정**: 기관의 이미지를 실질적으로 저하시키는 내용
     * 비교 문장에서도 대상 기관에 대한 부정적 평가는 추출
     * 예: "한국전력은 삼성전자보다 효율성이 낮다" → "효율성 낮음" (부정) 추출
   - **중립**: 기관과 관련되나 평판에 영향이 없는 단순 사실(일정, 통계, 공고 등)

5. **키워드 형태**: 문맥을 반영한 구체적인 표현(예: "혁신적 성과", "예산 낭비", "효율성 낮음")으로 추출해라.

6. **언어**: JSON의 모든 값은 반드시 한국어로 작성해라.

### [대상 기관]
이름: [organization]
(동의어: [synonyms])

### [기사]
[contents]

### [출력 형식]
반드시 아래 JSON 포맷으로만 응답해라. 주석, 설명은 절대 포함하지 마라.

{{
    "positiveKeywords": ["긍정 키워드1", "긍정 키워드2", ...],
    "negativeKeywords": ["부정 키워드1", "부정 키워드2", ...],
    "neutralKeywords": ["중립 키워드1", "중립 키워드2", ...]
}}
"""
```

**핵심 변경점**:
1. ✅ Line 1에서 "대상 기관 중심 추출" 명시
2. ✅ 추출해야 하는 경우와 안 하는 경우를 구체적으로 나열
3. ✅ 비교 문장 처리 섹션 추가 (Line 2)
4. ✅ 비교 문장 예시 추가

## 🎯 최종 권장: 개선안 1 (문장 구조 분석 중심)

### 권장 이유
