# 기관/카테고리 자동 추론 가이드

## 📋 요약

**이제 URL만 입력하면 기관과 카테고리를 자동으로 추론합니다!**

---

## 🎯 답변: 선택사항입니다!

### ✅ 방법 1: URL만 입력 (완전 자동) - **가장 간단!**

URL에 도메인 정보가 있으면 기관과 카테고리를 자동으로 추론합니다.

```bash
# 기관/카테고리 모두 자동 추론
python3 test_single_article.py \
  --url "https://www.motie.go.kr/kor/article/ATCL2826a2625/69968/view"

# 실행 시 자동으로:
# ✅ URL에서 기관 자동 추론: A0001 (산업통상자원부)
# ✅ URL에서 카테고리 자동 추론: B0002 (공고 키워드 발견)
```

**자동 추론 기능:**
- 🔍 도메인에서 기관 자동 인식 (motie.go.kr → 산업통상자원부)
- 🔍 URL 패턴에서 카테고리 추론 (공고/보도/정책 키워드)

---

### ✅ 방법 2: 직접 지정 (정확한 제어)

자동 추론이 실패하거나 직접 지정하고 싶을 때:

```bash
python3 test_single_article.py \
  --url "https://example.com/article" \
  --org "A0001" \
  --category "B0002"
```

---

### ✅ 방법 3: 원문 입력 시 (필수 지정)

원문을 직접 입력할 때는 URL 정보가 없으므로 반드시 지정해야 합니다:

```bash
python3 test_single_article.py \
  --text "기사 원문..." \
  --org "A0001" \
  --category "B0002"
```

## 💡 추론 규칙 커스터마이징

필요하면 `test_single_article.py` 파일의 추론 규칙을 수정할 수 있습니다:

### 기관 추론 규칙 수정

```python
# test_single_article.py의 infer_org_from_url() 함수
org_mappings = {
    'motie.go.kr': 'A0001',  # 산업통상자원부
    'msit.go.kr': 'A0003',   # 과학기술정보통신부
    # 여기에 새 도메인 추가
    'your-domain.go.kr': 'A0099',
}
```

### 카테고리 추론 규칙 수정

```python
# test_single_article.py의 infer_category_from_url() 함수
if any(keyword in url_lower for keyword in ['news', 'press', '보도']):
    return 'B0001'  # 보도자료
# 여기에 새 패턴 추가
elif 'your-keyword' in url_lower:
    return 'B0099'
```
